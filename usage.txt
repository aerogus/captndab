```bash

```

multicast limité à la boucle locale

```bash
sudo route add 239.0.0.0/8 127.0.0.1
sudo route delete 239.0.0.0/8 127.0.0.1
```

check routage

```
netstat -nr -f inet | grep 239.0.0
```
